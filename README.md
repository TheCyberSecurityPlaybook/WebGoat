# WebGoat

I am completing a web application security course using WebGoat, a deliberately insecure Java application for practicing detection and exploitation of common vulnerabilities in real open‑source components. This repository contains my walkthrough notes, exercise solutions, and proof‑of‑concepts (PoCs). Purpose: learn vulnerabilities (XSS, SQLi, CSRF, auth flaws, etc.), improve secure development and testing skills, and document reproduction steps and remediation advice.

All work is for educational purposes only; do not use these findings for malicious activity.
